# TyCode - NPM Package

Install TyCode as a global npm package for easy access from anywhere.

## Installation

```bash
npm install -g @eronic-company/tycode
```

This will:
1. Download the pre-built binary for your platform (Linux, macOS, or Windows)
2. Install it as a global command

## Usage

Once installed, simply run:

```bash
tycode
```

## Supported Platforms

- **Linux**: x86_64, aarch64
- **macOS**: x86_64 (Intel), aarch64 (Apple Silicon)
- **Windows**: x86_64

## Updating

To update to the latest version:

```bash
npm install -g @eronic-company/tycode@latest
```

## Uninstalling

```bash
npm uninstall -g @eronic-company/tycode
```

## Configuration

TyCode uses the same configuration as the standalone version. Settings are stored in `~/.tycode/config.json`

See the main [README](README.md) for detailed configuration and usage instructions.
